Upload-Integration

Notion Upload

Studiendaten werden nach jeder abgeschlossenen Studie automatisch in Notion hochgeladen. Speichern schreibt den notion-Block in hardware_config.json — danach Server neu starten.

Schritt-für-Schritt

Notion Setup

1
Integration erstellen

Öffne notion.so/profile/integrations → New integration.
Name: z.B. „StudyRunner". Typ: Internal.
Capabilities: Read content, Update content, Insert content.

2
API Key kopieren

Auf der Integration-Seite unter Secrets → Token kopieren.
Sieht aus wie: secret_abc123…
→ In das Feld API Key eintragen.

3
Notion-Seite erstellen & Integration einladen

Erstelle eine neue leere Seite in Notion (z.B. „StudyRunner Data").
Oben rechts: Share → Invite → Integration auswählen → Full access.

4
Parent Page ID herausfinden

Die Seite in Notion öffnen → URL ansehen:
notion.so/MeinWorkspace/AbcDef1234567890abcdef1234567890
Die letzten 32 Zeichen (ohne Bindestriche) sind die Page ID.
Alternativ: Share → Copy link → die ID steht am Ende der URL.

5
Database ID (optional)

Leer lassen → beim ersten Upload wird automatisch eine Datenbank
„StudyRunner Participants" mit dem richtigen Schema erstellt.
Falls du eine bestehende DB verwenden willst: DB in Notion öffnen →
URL: notion.so/database-id?v=… — die 32 Zeichen vor ?v=.

Datenbankschema (falls manuell erstellt)
